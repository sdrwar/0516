■파이썬
-프로그래밍 언어 중 하나 : 숫자와 영어를 이용하여 구성

파이썬 연습을 할 때에는 주피터 노트북(랩) 사용(프롬프트 창에서 작업하기엔 번거로우므로)
프로젝트를 할 때에는 VS code를 사용하여 작업

-주피터 설치 시 cmd창에 pip install jupyter를 입력 
->pip install 명령어는 설치 시 자주 사용하게 되는 명령어니 기억해두기(검증이 되지 않은 패키지는 설치하면 안된다)

프로젝트 같은 작업을 할 때에는 폴더를 새로 만들어서 하기(현재는 윈도우 탐색기 있는 곳에서 해도 됨)

※명령 프롬프트 창을 닫으면 주피터 노트북 창도 사용할 수 없다 (centos와 ssh접속을 한 cmd창 처럼)

in [*] : *이라는 표시가 나오면 계속 동작 중 이라는 뜻이기 때문에, 빠르게 멈춰줘야 한다

마크다운 : 실행시키는 것이 아니라 문서 작업같은 기록을 할 때 쓰인다?

m으로 마크다운 모드 y로 복구
-
가상환경 : 각각의 도구들이 따로 설치된 폴더 등의 환경 (방 1의 각각의 재료와 도구로 만들 수 있는 품목과 방 2의 재료로 만들 수 있는 품목이 도구와 재료에 따라
다르다)
-
파이썬 학습
1. 문법
 -자료 형태
 -연산(계산)기호 (+. -, *, /...)
 -변수*
 -제어문(조건/반복)
2. 도구 사용법
 -파이썬에서 제공하는 기본 도구
 -외부에서 설치 해야하는 도구(pip install로 패키지 다운로드, pypi.org에서 패키지를 확인할 수 있다)
-
변수
1. 긴 값을 간결하게 사용
2. 값을 이어서 사용

자료 형태
 - 변수에 대입되는 값 형태
 - 파이썬의 기본 자료 형태 7개
 - 숫자, 문자, 논리, 딕셔너리, 리스트 <- 필수 암기해야 하는 파이썬의 기본
   튜플, 셋(집합)
 - 기본이 아닌 자료 형태 (무한대)

변수로 쓰일 땐 snake case 방식으로 자주 쓴다 (언더바 _로 쓰는 변수?)

숫자에 관련된 함수는 암기하는 것이 좋다 (int, float)
-
문자 연산 중 indexing 중요
indexing []: 한 글자 추출하기
slicing [:]: 여러 개 글자 추출하기

공통 사항 : 시작위치는 0, 끝위치는 +1 상태
-
문자 함수 관련
문자 기능 (숫자는 찾기 기능이 없다, 숫자는 아래 기능을 할 수 없다, 연산은 가능)
1. 개수 - count()
2. 찾기 - find()
3. 변환 - replace()
4. 잘라내기 - [:] slicing, split()
5. 합치기 - join()
6. 전체길이 - len()
-
find 함수 사용 시 뒤의 숫자는 찾을 문자의 시작위치
