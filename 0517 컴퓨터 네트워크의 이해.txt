[네트워크의 기본 : 클라우드의 연결 (클라우드 위에 있는 네트워크)]

클라우드의 보안 및 관리 : 네트워크를 통해

클라우드에 접속하기 위한 기초 : 네트워크

*네트워크, 데이터베이스* 아는게 중요

-컴퓨터 네트워크의 이해-

1. 컴퓨터 네트워크의 기초 : IP의 목적과 이유를 알기위해서

2. 네트워크의 종류: 컴퓨터 네트워크, 휴먼 네트워크
*네트워크의 목적 : 정보를 전달하고, 정보를 얻기 위함.

3. 컴퓨터 네트워크의 구성 : 단말기, 전송장치, 연결하기 위한 선로의 구성
*최초의 컴퓨터의 단말기 구성 : PC와 프린터
-그 이후, 와이파이 공유기, 셋톱박스, 냉장고 등의 단말기가 추가됨. (IOP 사물 인터넷?)

4. 스위치 : PC와 PC간의 연결을 위한 도구 (여러대의 PC를 효율적으로 통신할 수 있게 해주는 도구)

5. 라우터 : 위치를 전달해 주는 역할 (길을 찾아주는 장비)
-최초의 역할: 군사목적

상용화
1980년대 : 각 PC의 OS언어를 번역해주어, 각각의 PC에서의 연결을 도와주는 목적으로 만들어짐
(그 당시 각 PC의 OS가 다르면, 이메일도 주고 받을 수 없었다.)

-라우터 : 네트워크(스위치)와 네트워크(스위치)를 연결해주는 인터 네트워크
라우터계의 1위기업 : 시스코
